CSS Combinators Reference

Combinators

Selector Example description
descendant selector (space) The descendant selector matches all elements that are descendants of a specified element
child selector (>) The child selector matches all elements that are children of a specified element
adjacent sibling selector (+) The adjacent sibling selector matches all elements that are the adjacent siblings of a specified element
general sibling selector (~) The general sibling selector matches all elements that are siblings of a specified element
